# AgroSense

Projeto desenvolvido para monitorar a umidade do solo com Arduino e sensor capacitivo. A solução realiza a leitura contínua dos dados, converte os valores em porcentagem e apresenta uma visualização mais clara para apoiar decisões de irrigação.

## O que tem neste repositório até o momento:
- Código da lógica utilizada no Arduino
- Lógica da calculadora financeira
- Script do banco de dados

## Tecnologias até o momento:
Arduino IDE, Arduino, sensor capacitivo de umidade do solo, GitHub, HTML & JavaScript e MySQL.

## Regras de Condução do Projeto:

•	Atas e Reuniões:
Foram estabelecidos como horários de reunião encontros presenciais todas as quintas-feiras, no período após o término das aulas;

Também foram determinadas reuniões online na plataforma Discord, realizadas as terças-feiras no horário das 19:00, a fim de que sejam feitos Sprints Reviews do trabalho.

Todas as informações serão registradas em documento para melhor versionamento do projeto;

•	Consequências:
Caso haja falta de alguns dos integrantes entre as reuniões estabelecidas, a mesma informação será registrada em nosso documento;

As decisões feitas durante estes encontros, não poderão ser modificadas por algum integrante que não estiver presente no dia;

Em caso de atraso, o integrante deverá iniciar e apresentar seu andamento no projeto para os demais;

Mediante a atrasos e faltas, o integrante será redirecionado para escrever e detalhar todas as informações coletadas durante a reunião em nossa Ata;

Referente a falta de entrega nas datas previstas, será documentado e contado como indicador de participação do projeto. Também serão avaliadas se houver justificativas plausíveis, com adendo a observações;
